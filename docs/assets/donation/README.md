# Donation Assets

Place the payment QR code images here:

- `wechat-pay-reward.jpg`: WeChat Pay reward QR code.
- `alipay-reward.jpg`: Alipay reward QR code.

Use the original images exported from WeChat Pay and Alipay. Do not redraw,
compress aggressively, crop, or regenerate the QR codes, otherwise they may fail
to scan or receive payment.
