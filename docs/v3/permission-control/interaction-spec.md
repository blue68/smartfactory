# 权限控制模块 Interaction Spec

## 1. 交互总则

1. 所有可写操作必须有明确反馈：成功、失败、处理中、已撤销。
2. 所有高风险操作必须二次确认：停用、删除、批量覆盖、重置密码、重置授权。
3. 所有授权保存必须体现“保存前摘要”和“保存后生效范围”。
4. 页面级权限不足时，不展示入口；接口级权限不足时返回错误，不做静默失败。

## 2. 页面通用状态

### 2.1 加载态

- 首次进入页面展示骨架屏或表格加载态。
- 左侧树/列表加载时保留上一次选中态直至新数据返回，避免界面抖动。
- 授权页加载角色详情时，右侧面板先展示“正在加载授权信息”。

### 2.2 空态

- 列表为空时显示“暂无数据”+ 主操作按钮。
- 树为空时显示“暂无菜单，请先新建一级菜单”。
- 人员无角色时显示“尚未分配角色”并引导进入分配页。

### 2.3 错态

- 查询错误：顶部或卡片内显示错误条，并保留重试按钮。
- 保存错误：表单底部显示错误说明，不自动清空表单。
- 授权保存失败：保留已勾选状态，提示接口失败原因。

### 2.4 禁用态

- 已停用租户：所有编辑与新增按钮置灰，仅允许查看。
- 已停用角色：不可继续分配给人员。
- 已禁用人员：不可登录，重置密码按钮可选是否保留由产品决定，但默认可见。

## 3. 租户配置交互

### 3.1 新建/编辑

- 点击 `新建租户` 打开弹框。
- 租户编码输入框失焦后做唯一性校验。
- 套餐切换后自动刷新功能开关建议值。
- 提交成功后关闭弹框并刷新列表。

### 3.2 停用

- 点击停用弹出确认框，提示“将影响当前在线会话”。
- 确认后立即刷新租户状态与统计卡。

## 4. 菜单与功能交互

### 4.1 树选择

- 单击菜单节点，右侧同步加载菜单详情和功能点列表。
- 双击或回车可进入编辑，单击仅选择不编辑。
- 展开/收起状态应记忆到本次会话。

### 4.2 新增与排序

- 新增子菜单时自动带出父节点分组和层级。
- 保存排序前显示“将影响 X 个节点”的提示。
- 系统预置菜单不允许删除，只允许停用。

## 5. 角色配置交互

### 5.1 新建/复制

- `复制系统角色` 会预填编码、菜单授权和默认数据范围。
- 创建后默认跳转到角色授权页。

### 5.2 禁用/删除

- 禁用前显示当前绑定人数。
- 若角色仍被人员引用，删除按钮不可用或转为“解除绑定后删除”。

## 6. 人员配置交互

### 6.1 新建/编辑

- 支持批量导入和单个新建两种入口。
- 新建成功后弹出“是否立即分配角色”的引导。

### 6.2 启停

- 禁用人员前提示“禁用后不可登录”。
- 若为唯一租户管理员，则禁用按钮需要额外确认并提示替代管理员。

## 7. 角色授权交互

### 7.1 授权树

- 勾选一级菜单时，可联动勾选其下默认功能点。
- 取消菜单时，自动清除该菜单下所有功能点。
- 支持搜索菜单编码/名称定位节点。
- 支持“仅看已授权”筛选。

### 7.2 数据范围

- 数据范围选择 `全部` 后，附加限定项自动清空。
- 选择 `指定仓库/客户/供应商` 时，必须至少选择一项。
- 若当前系统没有部门模型，“本部门”需要显示为受限状态或提示待接入组织中心。

### 7.3 保存

- 保存前显示授权摘要：菜单数、功能点数、数据范围。
- 保存后立即刷新角色详情，并可弹出“已生效”提示。

## 8. 人员角色分配交互

### 8.1 分配方式

- 支持单人分配和批量分配。
- 支持追加与覆盖两种模式。
- 主角色必须包含在已选角色内。

### 8.2 生效时间

- 若填写失效时间，必须晚于生效时间。
- 过期角色在列表中显示“即将失效”或“已失效”状态。

## 9. 批量操作交互

批量操作统一规则：

- 未勾选任何对象时按钮置灰。
- 点击后先展示影响摘要，再确认提交。
- 成功后返回结果列表，失败对象单独列出失败原因。

适用操作：

- 批量启用/停用租户
- 批量启用/停用菜单
- 批量禁用/启用角色
- 批量禁用/启用人员
- 批量分配角色
- 批量取消授权

## 10. 危险操作确认

以下操作必须二次确认：

- 删除租户
- 停用租户
- 删除菜单节点
- 删除角色
- 禁用人员
- 覆盖分配角色
- 重置密码
- 清空角色授权

确认文案规则：

- 标题明确对象，例如“停用租户”
- 说明影响范围，例如“将影响 18 名用户”
- 给出主操作按钮和取消按钮
- 危险按钮使用红色
