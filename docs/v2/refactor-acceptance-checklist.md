## 验收流程与门禁

```
P0-A 路径修复完成
  + P0-B DB Migration 执行完成
        ↓
      P1 Mock 替换验收
        ↓
      P2 UI 对齐验收
        ↓
      已正常模块回归验收
        ↓
      @senior-qa-engineer 输出测试报告
        ↓
      @senior-ai-agent-pm 最终产品验收
        ↓
      允许发布
```

任意 P0 条目未通过，禁止进入后续阶段。
任意 P1 条目有 Mock 数据残留，禁止进入 P2 阶段。
P2 设计稿对应关系未补全，禁止开始 UI 对比验收。
